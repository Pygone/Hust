Hust: A Python Tool for the Hust Platform

This repository contains a Python tool for interacting with the Hust platform. It includes functionalities such as logging in, scheduling courses, selecting badminton courts, topping up school cards, and more.

Dependencies

Python Dependencies

Install the required Python packages with pip:

pip install -r requirements.txt

OCR Dependencies

install tesseract and add it to the environment variable.

Usage

Logging In

Use the LoginSession class to log in to the Hust platform and get a user session.

from LoginSession import LoginSession userId = "xxx" password = "xxx" session = LoginSession(userId=userId, password=password)

Use the Operator class to perform operations on the Hust platform.

from operate import operator operation = operator(session, userId)

Course Schedule

Get the user's course schedule in JSON format.

course_info = operation.schedule() # course_info is a json object contains the course schedule of the user.

Course Score

Get the user's course score in JSON format.

course_score = operation.transcript() # course_score is a json object contains the course score of the user.

Automating Professional Course Selection

Automate the process of selecting a course. This is significantly faster than manual selection.

operation.course({"图神经网络": "万瑶"}, "09/20/18/00", "Attack")

Automating Badminton Court Selection

Automate the process of selecting a badminton court. This is significantly faster than manual selection. Partner is optional, if you want to specify your friend, you can add the partner info in the list in ["password", "name", "schoolId"] format or the script will choose your first partner you have set before.

operation.badminton("2021-09-20", "08", 1, partner=["password", "name", "schoolId"]) operation.badminton("2021-09-20", "08", 1) # both can work well

Professional Course Credit

Get the user's professional course credit.

course_credit = operation.professional_credit()

Automating Public Course Selection

Automate the process of selecting a public course. This is significantly faster than manual selection.

operation.public_course(["节奏世界"])
