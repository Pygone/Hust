Hust: 一个用于Hust平台的Python工具

此仓库包含一个用于与Hust平台交互的Python工具。它包括登录、排课、选择羽毛球场地、充值校园卡等功能。

依赖

Python依赖

使用pip安装所需的Python包：

pip install -r requirements.txt

OCR依赖

安装tesseract并将其添加到环境变量中。

使用方法

登录

使用LoginSession类登录Hust平台并获取用户会话。

from LoginSession import LoginSession userId = "xxx" password = "xxx" session = LoginSession(userId=userId, password=password)

使用Operator类在Hust平台上执行操作。

from operate import operator operation = operator(session, userId)

课程表

以JSON格式获取用户的课程表。

course_info = operation.schedule() # course_info是一个json对象，包含用户的课程表。

课程成绩

以JSON格式获取用户的课程成绩。

course_score = operation.transcript() # course_score是一个json对象，包含用户的课程成绩。

自动选择专业课程

自动选择课程的过程。这比手动选择快得多。

operation.course({"图神经网络": "万瑶"}, "09/20/18/00", "Attack")

自动选择羽毛球场地

自动选择羽毛球场地的过程。这比手动选择快得多。伙伴是可选的，如果你想指定你的朋友，你可以在列表中添加伙伴信息，格式为["密码", "姓名", "学校ID"]，或者脚本会选择你之前设置的第一个伙伴。

operation.badminton("2021-09-20", "08", 1, partner=["password", "name", "schoolId"]) operation.badminton("2021-09-20", "08", 1) # 两者都可以正常工作

专业课程学分

获取用户的专业课程学分。

course_credit = operation.professional_credit()

自动选择公共课程

自动选择公共课程的过程。这比手动选择快得多。

operation.public_course(["节奏世界"])
